# Demo — El Var Food and Coffee

Propuesta / demo de **menú digital** para el restaurante futbolero **El Var
Food and Coffee** (Urb. Santa Cecilia, Valencia, Edo. Carabobo — Venezuela).

## Qué es

Una página **autónoma de una sola pieza** (`index.html`) que sirve como muestra
comercial de cómo se vería la capa pública de este cliente sobre la plataforma
white-label.

- **Tema:** transmisión de estadio + cancha de fútbol (juega con "VAR").
- **Contenido:** estudiado de las publicaciones públicas de
  [@elvarfoodandcoffee](https://instagram.com/elvarfoodandcoffee) — especiales
  futboleros (Tiro Libre / Golazo / Penalti), parrillas, pizzería, café y
  agenda de partidos.
- **Interacción:** filtros de menú por "posición", marcador en vivo de ambiente,
  tema claro/oscuro y botones de pedido (en modo demo).

## Reglas respetadas

- ✅ **No toca Brotherhood.** Vive aislado en `demo/el-var/`; no modifica
  `src/`, la marca, el tema global ni los archivos compartidos del panel
  privado/público.
- ✅ **No se conecta a Supabase.** 100 % estático: todos los datos del menú y la
  agenda están embebidos en el propio HTML. Sin llamadas a `/api` ni a la base.

## Cómo verlo

Abre `index.html` en el navegador, o publícalo como Artifact.
Precios de referencia con fines de demostración.
